\subsubsection*{template$<$typename T$>$\newline
class Napi\+::\+Class\+Property\+Descriptor$<$ T $>$}
Property descriptor for use with {\ttfamily Object\+Wrap\+::\+Define\+Class()}.

This is different from the standalone {\ttfamily \doxylink{class_napi_1_1_property_descriptor}{Property\+Descriptor}} because it is specific to each {\ttfamily \doxylink{class_napi_1_1_object_wrap}{Object\+Wrap}\texorpdfstring{$<$}{<}T\texorpdfstring{$>$}{>}} subclass. This prevents using descriptors from a different class when defining a new class (preventing the callbacks from having incorrect {\ttfamily this} pointers).
