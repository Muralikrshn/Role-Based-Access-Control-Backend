class Napi\+::\+Maybe$<$ T $>$}
A simple \doxylink{class_napi_1_1_maybe}{Maybe} type, representing an object which may or may not have a value.

If an API method returns a Maybe$<$$>$, the API method can potentially fail either because an exception is thrown, or because an exception is pending, e.\+g. because a previous API call threw an exception that hasn\textquotesingle{}t been caught yet. In that case, a "{}\+Nothing"{} value is returned.
